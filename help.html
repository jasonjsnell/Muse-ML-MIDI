Getting Started

Connecting Your Browser to Ableton

Follow these instructions to connect your browser to Ableton (or another DAW).

You must use Chrome or another browser that has Web Bluetooth.

MIDI Mapping in Ableton

Follow these instructions to map the brainwave MIDI CC's to music parameters inside Ableton. Instead of turning a knob on a MIDI controller, you'll press the "Map" button next to a State. That will send a single MIDI CC command over to Ableton, which can receive it if it's in MIDI Map Mode.

IMPORTANT: If the Muse headset is connected and streaming data, it's nearly impossible to do MIDI mapping because Ableton will be trying to map several MIDI CC streams simultaneously. It's much easier to do if the headband is off and you press the Map buttons.

Connecting to the Muse headset

  1. Turn on the Muse headset
  2. Open the creator tool in your browser
  3. Press the "Connect" button. A Web Bluetooth window will open.
  4. Wait for your Muse to show up in the window. Select it and click "Pair."

Your brainwave (and with the Muse 2, your heartbeat) data will start to stream in. The numbers in the grey box will change and you'll see a white line showing your brainwave frequency spectrum, ranging from Delta on the left to Gamma on the right.

EEG Streaming Data

The incoming data stream from the EEG headset is analyzed by ML5 (machine learning) and assessed. The incoming signal could be determined to be:

Noise

This is when the headband is not on someone's head, or the connection is loose. I usually use this MIDI CC to pull down the volumes of the sounds so the atmospheric electricity isn't sonified. I only want to have music output when someone has the headset on with a good fit.

Muscle Tension

This category includes tension in the forehead, jaw clenches, eye blinks. The Muse headset can acts as an EMG (electromyography = muscle) as well as an EEG, and the biometric data of a person's muscle tension can be turned into a sound.

Relaxed focus

This is a new training via ML. I've found this state activates when my face is still and I'm reading or focusing on something, like reading text on my computer screen.

Clear mind

When I unfocus my eyes and blank out my mind, all the electrical activity of all the brainwaves drops, and this state is detected.

Meditative mind

The Muse headset is particularly good at detecting elevated alpha, which usually indicates the eyes are closed and the mind is becoming quiet.

Dreaming

If very tired, or meditating long enough that you start to have dream flashes, this state activates.

Dominant State

The state that the ML determines is the dominant state will have an arrow pointing at it. For example:

> CLEAR

Biometric Data

Below the ML states are readouts of the 5 brainwaves, your heart BPM (if you're using the Muse 2 or later) and accelerometer / movement data.

MIDI Controls

Each mental state (like "Clear Mind" or "Focused") has MIDI control options for both CC (Continuous Control) and Notes.

Note Toggle

Checking the Note box enables MIDI note output when that state becomes dominant. This can be used to trigger samples, start loops, or play instruments.

Note Map Button

Clicking the Map button next to a Note will send a one-time MIDI note signal. Use this to map a state to a sampler or clip in your DAW (like Ableton).

CC Stream Toggle

Checking the CC box starts a continuous stream of MIDI CC values (0â€“127) based on how active that state is. These streams can modulate parameters like filter cutoff, reverb amount, or visual effects.

CC Map Button

Clicking the Map button next to a CC sends a single CC value (based on the current brain state) to your DAW. This makes it easy to MIDI map without overwhelming the DAW with a full data stream.

Tip: Disable streaming before mapping, or remove the headset temporarily, so only one MIDI signal is sent during mapping.
